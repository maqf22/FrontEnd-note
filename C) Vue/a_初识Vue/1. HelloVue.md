# 一、Vue Cli

安装
```shell
npm install -g @vue/cli
yarn global add @vue/cli
```

创建项目
```shell
vue create my-project
vue ui
```


# 二、单文件组件

- 提供了更好的封装性
- 优雅的模板支持
- scoped CSS支持
- 通过vue-loader可以配合各种预处理器进行构建


# 三、命名规则

camelCase: videoExampleComponent
PascalCase: VideoExampleComponent
kebab-case: video-example-component
